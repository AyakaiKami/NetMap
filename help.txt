    1. get <propriety> <type> <ident>                               (done)

                    RAM

                    interface (not yet) 
                    state
                    load
        Type : id
               ip
               name
    2. list                                                         (done)
        list IDs of all active VMs on the device
    3. parabol                                                      (done)
        lists save identification and data of the save
        (Lateralus : Parabol from Tool) 
    4. parabola <ID_SAVE>                                           (done)
        with function (close parabola)
        draws the graph associated with a save 
        (Lateralus : Parabola from Tool)
    5.hexagram                                                      (done)
        with function (close hexagram)
        draws the current graph changes every T minutes
        (Deftones : Hexagram from Deftones)
    6.flash <save_identification> <ID>                              (done)
        prints all data on a VM previously saved
